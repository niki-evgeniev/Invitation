# 🎉 Baptism Invitation Web Application

A full-stack web application for creating and sharing personalized baptism invitations, built with **React** (frontend) and **Spring Boot** (backend).

🔗 **Live Demo:** https://toncho.online

---

## 🚀 Features

* 📝 Create custom baptism invitations
* 🎨 Clean and modern UI with responsive design
* 📱 Mobile-friendly (fully responsive)
* 🔗 Share invitations via unique link
* ⚙️ RESTful API for managing invitations
* 🗄️ Persistent data storage using MySQL

---

## 🛠️ Tech Stack

### Frontend

* React
* JavaScript (ES6+)
* HTML5 / CSS3

### Backend

* Spring Boot
* REST API
* Maven

* MySQL
